To Run Any Script Files available in Services menu, Client's Machine should have softwares like :

  • Python 2.0 or higher version.
  • Nmap v7.91
Steps to install Python v2.0 or Higher version :

Step 1 : Visit the Official Python website to download the Python

              Python Official Website  

Step 2 : Select the proper version intended for your Operating System.

  • Windows OS :       Python for Windows   Python for Windows v3.9.4

  • Linux based OS :   Python for Linux

Step 3 : Make sure to check the Add Python -v to PATH option while installing python.



{% comment %} Nmap installation {% endcomment %}
Steps to install Nmap v7.91 or Higher version :

Step 1 : Visit the official website of Nmap Installer to download nmap v7.91

  Nmap  

Step 2 : Select the proper version intended for your Operating System.

  • Windows OS :    Windows OS Nmap v7.91

  • Linux based OS :   Nmap for Linux

Step 3 : Make sure to add Nmap folder location address in Environmental Variable in Windows OS to PATH option while installing nmap, if not added automatically.

    Steps to add a path variable in Windows OS :

  • Step 1 : Go to This PC, on Desktop
  • Step 2 : Right Click -> Properties -> Advanced System Settings (Left side of the screen)
  • Step 3 : Click on Environment Variable -> Locate 'path' or 'PATH' variable in User Variables and System Variables.
  • Step 4 : Click on path and click on Edit. Here path for Python would aleardy be added, now we need to add the nmap path
  • Step 5 : Locate the nmap folder in your local system(C: -> Program Files or Program Files(x86) -> Nmap)
  • Step 6 : Copy the path and paste in edit section of Environmenat Variables



    • Now Scripts will run smoothly.
